# Public Assets

## 需要的图标文件

请添加以下图标文件到此目录：

- `icon-192.png` - 192x192px PWA 图标
- `icon-512.png` - 512x512px PWA 图标
- `apple-touch-icon.png` - 180x180px iOS 图标
- `favicon.ico` - 网站图标

## 临时使用

在开发阶段，你可以从原项目复制图标，或使用在线工具生成：
- https://realfavicongenerator.net/
- https://favicon.io/

## 推荐尺寸

- **icon-192.png**: 192x192px (PWA 最小尺寸)
- **icon-512.png**: 512x512px (PWA 推荐尺寸)
- **apple-touch-icon.png**: 180x180px (iOS 设备)
- **favicon.ico**: 32x32px (浏览器标签页)
